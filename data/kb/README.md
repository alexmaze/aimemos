# RAG 模块示例数据

本目录用于存放测试文档，用于验证 RAG 模块的功能。

## 使用方法

1. 在此目录下添加 `.txt` 或 `.md` 文件
2. 运行索引脚本：
   ```bash
   cd rag
   python ingest.py --data-dir ../data/kb --kb-id test_kb
   ```

## 示例文档

您可以创建以下示例文档来测试：

```bash
# 创建示例文档
echo "人工智能（Artificial Intelligence，简称AI）是计算机科学的一个分支，致力于创建能够执行通常需要人类智能的任务的系统。" > ai.txt

echo "机器学习是人工智能的一个子领域，它使计算机系统能够通过经验自动改进性能，而无需显式编程。" > ml.txt

echo "深度学习是机器学习的一种方法，它使用多层人工神经网络来学习数据的表示。深度学习在图像识别、语音识别和自然语言处理等领域取得了突破性进展。" > dl.txt
```

## 注意事项

- 支持的文件格式：`.txt`, `.md`
- 文件应使用 UTF-8 编码
- 建议每个文件不超过 100KB
- 文件内容应为纯文本或 Markdown 格式
